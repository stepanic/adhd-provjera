# ADHD provjera — arhivirano

> [`domovinatv/provjera.domovina.ai`](https://github.com/domovinatv/provjera.domovina.ai),
> a produkcijska aplikacija je dostupna na <https://provjera.domovina.ai>.

GitHub Pages na ovoj lokaciji
([`stepanic.github.io/adhd-provjera`](https://stepanic.github.io/adhd-provjera/))
servira samo redirect stranicu prema novoj lokaciji. Cjelokupna povijest commit-ova
ostaje sačuvana ovdje radi reference, ali za nastavak razvoja, prijave problema i
nove verzije aplikacije idite na novi repozitorij.

## Kontekst

ADHD provjera je započela kao osobni projekt korisnika `stepanic` 2026. godine
kao offline PWA za samoprocjenu simptoma ADHD-a u odraslih prema upitniku ASRS
v1.1 (WHO / Harvard Medical School). Tijekom razvoja se preselila pod jedinstveni
brand **Domovina Provjera** — obitelj alata za samoprocjenu mentalnog zdravlja u
Domovina ekosustavu (uz `domovina.tv`, `domovina.ai`, `pay.domovina.ai`,
`gis.domovina.ai`, i drugi).

Selidba u `domovinatv` organizaciju i na poddomenu `provjera.domovina.ai` označava
službeni prelazak iz personalnog projekta u javni brandirani modul. Sva
funkcionalnost je zadržana 1:1 (ASRS v1.1 screener, kratka i puna verzija,
ilustrativni primjeri, offline obrada).
